# 📄 Sample Resumes for RecruitFlow

This directory contains pre-configured sample resume documents you can use to test the RecruitFlow AI ingestion pipeline.

## 📂 Available Files

1. **[sample_cv_backend_engineer.pdf](file:///d:/Innovator%20Challenge/resumes/samples/sample_cv_backend_engineer.pdf)**:
   - A basic valid PDF file representing a Backend Engineer.
   - Ideal for testing PDF uploads.

2. **[sample_cv_frontend_engineer.docx](file:///d:/Innovator%20Challenge/resumes/samples/sample_cv_frontend_engineer.docx)**:
   - A fully valid Word Document (`.docx`) file containing detailed text for a Frontend Engineer (Jane Doe).
   - Ideal for testing DOCX file parsing and text extraction.

## 🚀 How to Use

1. Navigate to the Candidate Application Form at the root page `/`.
2. Enter the applicant details (e.g. name, email).
3. Drag and drop or browse to select one of these sample files.
4. Click **Submit Application** to observe real-time AI parsing and status routing!
